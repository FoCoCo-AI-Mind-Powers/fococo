FoCoCo | Website Structure and Style 
GLOBAL STYLE: 
• Font: Montserrat 
• Background: Pure black 
• Edge Glow Lines: FoCoCo Blue #4285F4 (subtle) 
• Arcs: Only inside the png images 
• Spacing: Cinematic, premium, airy 
• Images: Use the 6 provided png’s (no edits) 
• Screenshot Width Max: 480–520px at desktop, 80% width on mobile. 
• Max Width for Text: ~720px 
• Section Titles: H2 
• Labels: H3 
• Hero only: H1 
• No extra graphics or effects 
SCREENSHOTS:
• Always use real UI images 
• Displayed as “floating” 
• On black background 
• With thin FoCoCo-color rim glow (blue 60% opacity) 
• Slight tilt (2–5 degrees) for premium depth 
• Soft shadow, not harsh 
• Always with breathing room 
• Always vertically aligned under text, not beside images 
• Never use full width 
• Never use busy grids
